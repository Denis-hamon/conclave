# Code of Conduct

Conclave adopts the [Contributor Covenant, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its Code of Conduct. All participants — maintainers, contributors, and users interacting in the project's spaces — are expected to abide by it.

## Scope

This Code of Conduct applies to all Conclave project spaces (GitHub issues, pull requests, discussions, code review, and any future community channels) and to interactions that represent the project publicly.

## Reporting

If you observe or experience behavior that violates this Code of Conduct, please report it privately to the maintainer at **denis.hamon1@gmail.com**. Reports are handled confidentially. See `SECURITY.md` for the separate channel used to report security vulnerabilities.

## Enforcement

Maintainers are responsible for clarifying and enforcing standards. They may remove, edit, or reject contributions that do not align with this Code of Conduct, and may temporarily or permanently restrict participation for behavior they deem inappropriate.

The full text of the Contributor Covenant, including the community impact guidelines used to determine consequences, is available at the link above.
